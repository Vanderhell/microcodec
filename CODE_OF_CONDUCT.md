# Code of Conduct

## Our standard

Participation in this project should remain respectful, direct, and
constructive.

Examples of expected behavior:

- discussing technical tradeoffs in good faith
- giving actionable review feedback
- focusing on code, behavior, and evidence
- respecting contributor time and context

Examples of unacceptable behavior:

- harassment or personal attacks
- abusive language
- bad-faith review behavior
- deliberate disruption of project discussions

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of
conduct. Unacceptable behavior may result in warnings, removal of content, or
temporary or permanent bans from project spaces.
